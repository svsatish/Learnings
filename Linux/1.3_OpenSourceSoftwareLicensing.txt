1.3 Open source Software and Licensing

Open source philosophy - freedom to copy, modify and distribute software
Permissive - some restrictions on license
Copyleft - more restrictions - must use same license

—————————

Open source licensing models

Public Domain > CCO 1.0 (Creative commons 1.0 Universal public domain dedication)
Permissive > BSD
Copyleft > GPL (GNU general public license)
